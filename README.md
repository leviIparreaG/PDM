# Tarea 2 - Programacion de Dispositivos Moviles

Repositorio de la Tarea 2 en Android Studio.

## Integrantes
- LeviIparreaG

## Descripcion
La app implementa dos pantallas con componentes graficos de Android:
- TextView / etiquetas
- EditText / cajas de texto
- CheckBox
- RadioButton
- Button
- ImageView

Pantallas incluidas:
1. `MainActivity`: captura de datos y opciones del pedido.
2. `PedidoActivity`: resumen, metodo de pago y confirmacion.

Se incluyen layouts para `portrait` y `landscape` (`layout/` y `layout-land/`) para adaptarse a rotacion y tamanos estandar.

## Ejecucion
1. Abrir el proyecto en Android Studio.
2. Sincronizar Gradle.
3. Ejecutar en emulador o dispositivo (`minSdk 24`).
