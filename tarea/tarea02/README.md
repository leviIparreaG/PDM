# Tarea 02 - Programacion de Dispositivos Moviles

## Descripcion

Aplicacion Android de un restaurante con 2 pantallas:
- **Pantalla 1 (Login):** Formulario de bienvenida con nombre, correo, telefono, RadioButtons (tipo de visita) y CheckBoxes.
- **Pantalla 2 (Pedidos):** Menu de productos con CheckBoxes, cantidades, modalidad de entrega (RadioButtons), Spinner y campo de comentarios.
